# Bellabeat Case Study - Google Data Analytics
Analysis of smart device fitness data to help Bellabeat's marketing strategy.

## 1. Ask
How do Bellabeat's customers use their smart devices? Identifying trends in physical activity and sleep.

## 2. Prepare
Data source: FitBit Fitness Tracker Data (CC0: Public Domain, via Kaggle).

## 3. Process & Analyze
Using **R** to clean the data and find correlations between daily steps and sleep quality.

## 4. Share (Visualizations)
![Daily Steps vs Sleep Quality](grafico_passos.png)

## 5. Act (Insights)
* **Insight 1:** Users with higher activity levels tend to have more consistent sleep patterns.
* **Insight 2:** Most users do not reach the 10,000 steps goal, suggesting a marketing opportunity for "activity reminders".
